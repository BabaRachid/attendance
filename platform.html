الغياب المباشر

الأستاذ يرسل من هاتفه — والغياب يظهر هنا فورًا

⚡ الربط المباشر (Firebase — مرة واحدة)
  1. افتح console.firebase.google.com وأنشئ مشروعًا (بدون Analytics).
  2. من القائمة: Build → Realtime Database → Create database → اختر test mode.
  3. من تبويب Rules اجعلها: {"rules":{".read":true,".write":true}} ثم Publish.
  4. ⚙ Project settings → أسفل الصفحة أيقونة الويب </> → سجّل التطبيق → انسخ firebaseConfig والصقه هنا:
🔗 رابط الأساتذة (ثابت لا يتغير)

الصق رابط صفحتك على GitHub Pages ثم انسخ الرابط النهائي وأرسله للأساتذة مرة واحدة فقط. أي تلميذ جديد أو إسناد أو إزالة تظهر عند الأستاذ تلقائيًا، وأي غياب يرسله يصل هنا لحظيًا.
